# 스프라이트 출처 메모

- Bulbagarden Archives `Category:Champions menu sprites`는 Pokémon Champions menu sprite 카테고리이며, 일반 메뉴 스프라이트 359개를 제공한다.
- 카테고리 페이지는 첫 200개와 다음 페이지로 나뉘어 표시된다.
- 샤이니 메뉴 스프라이트는 별도 카테고리 `Category:Champions Shiny menu sprites`에 있다.

원본 PNG는 지식팩에 직접 포함하지 않고, `optional_tools/download_champions_menu_sprites.py`로 사용자가 직접 내려받도록 한다.
